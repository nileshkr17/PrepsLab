Day 01 Assignment
MongoDB is a document-oriented database that is open-source and cross-platform. It was founded in 2009 and is currently owned and supported by MongoDB Inc. MongoDB is a popular choice for storing large amounts of data, as it is very scalable and flexible. It is also very easy to use, making it a good choice for developers of all skill levels. MongoDB uses a JSON-like format for storing data, which makes it easy to read and write. Data is stored in documents, which are collections of key-value pairs. Documents can be nested, and they can be linked to each other. This makes MongoDB a good choice for storing complex data structures.
Express.js is a popular back-end web application framework written in JavaScript. It is open-source and free to use, and it is used by millions of developers worldwide. Express.js is known for its simplicity, flexibility, and performance. It is easy to learn and use, and it can be used to create a wide variety of web applications. Express.js is built on top of the Node.js platform, which is a JavaScript runtime environment. Node.js is known for its speed and scalability, and it makes Express.js a very powerful and efficient web application framework.
React is a JavaScript library for building user interfaces. It is used to build single-page applications. React allows us to create reusable UI components. React is a declarative library, which means that you tell React what you want the UI to look like, and React figures out how to render it. This makes React code much easier to read and understand. React is also very efficient. It only renders the parts of the UI that have changed, which makes it very fast. React is a popular library, and there are a lot of resources available to help you learn it. There are also a lot of third-party libraries that can be used with React to add additional functionality.
Node.js is an open-source, cross-platform, back-end JavaScript runtime environment. Node.js uses an event-driven, non-blocking I/O model that makes it lightweight and efficient. Node.js is ideal for building real-time web applications that require high performance and scalability. Node.js is used by a wide variety of companies, including Netflix, Uber, and PayPal. It is also used by many open-source projects, such as Express.js, MongoDB, and AngularJS.
